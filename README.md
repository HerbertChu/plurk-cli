# Plurk-cli

## Made up with Deno runtime

### Start

```
deno run --allow-net src/Main.ts
```

### Test

```
deno test
```

### Install Denon

```
deno install -qAf --unstable https://deno.land/x/denon/denon.ts

```

### Start with Denon

```
denon start
```

## Dev. with VSCode

Adding `"deno.enable": true` to settings.json under .vscode is required
